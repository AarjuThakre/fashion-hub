# FashionHub Minimalist Boutique

A luxury, white-themed fullstack e-commerce clothing marketplace inspired by AJIO and Myntra, featuring visual order status timelines, review breakdowns, size guides, and similar product sliders.

## 🛠️ Features
- **Visual Order Progress:** Status nodes (`Ordered` ➔ `Shipped` ➔ `Delivered`) update dynamically in the dashboard.
- **Rating Breakdown Summaries:** Bar graphs showing rating distribution.
- **Apparel Size Chart Guide:** Grid modals displaying chest/waist sizes in inches.
- **Similar Product Recommendations:** Up to 4 category-matched garments on product details.
- **Express Shipping Add-ons:** Checkout checkboxes for express packaging and premium wrap.
